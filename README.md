# vuetify-sandbox

### Run Docker for development
```
docker-compose up --build
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
pm2 serve --spa dist 8080
```

### Lints and fixes files
```
npm run lint
```
